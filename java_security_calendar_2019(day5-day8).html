看到两个输入点，icons、filename均未做任何过滤，用户传入数这两个参数后，对其名字做判断，其中判断使用的是getName()方法，该方法仅能做简单../的过滤，如/../../../../hack.txt转成hack.txt。然后执行到关键代码String toDir = "/var/myapp/data/" + f_icons.getName() + "/";，发现toDir的值为/var/myapp/data/与f_icons.getName()拼接而成。此时通过../的形式对传入的值做控制，从而实现可以访问/var/myapp/目录下任意文件。
